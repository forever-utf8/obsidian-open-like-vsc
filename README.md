# Open Tabs Like VSCode

An Obsidian desktop plugin that brings VS Code style preview tabs to the file explorer.

## Features

- Single-click a note in the file explorer to open it as a preview tab.
- Double-click a note or its tab to pin it as a normal tab.

## Requirements

- Obsidian desktop `1.8.9` or newer

## Development

Install dependencies:

```bash
npm install
```

Build the plugin:

```bash
npm run build
```
